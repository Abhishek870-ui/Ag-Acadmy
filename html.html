    • What is Tag ?

    • Text palced between '<' and '>' is called as tag Eg <html>
    • Tags are classified into two types
      • Paired tags
      • Non-Paired tags

    Paired tags

  • The tags that have both opening and closing tags are called as paried tags.
  • The closing tags start with '/'

    • Non-Paired

  • The tags that have only opening tag but no closing tag are called as non-paired tags.
  • These tags are also called as
    1. Self closed tags
    2. Forcefully closed tags
    3. Empty tags
    4. Singular tags.
  • Example
    1. <br> or <br/> Break tag(line break)
    2. <hr> or <hr/> Hrizontal rule (h. line)
    3. <img> or <img> image tag

    HTTP ( Hypertext Transfer Protocol )

    It is used to enable/establish communication between client and server
    HTTP works as request-response protocol between client and server
    A web browser is client
    The computer who hosts the website is server

    HTTP request methods

    There are various methods of HTTP requests
    E.g. GET, POST, PUT, UPDATE, DELETE, CONNECT, … etc.
    Of which two methods are commonly used for request – response between client and server.
    Get method
    Post method

    Action Attribute

    This attribute is used to specify url of the server page to which we want to send our form data.

    Get Methods

    In this methods there is no security for data.
    This methods can send only limited data(2MB)
    It can carry row data from client to server.
    Notes:-
    GET requests can be cached.
    GET remain in browser history.
    GET requests can be bookmarked.
    GET requests should not deal with sensitive data.
    GET requests length is rest recited.
    GET requests have excellent performance
    GET requests cannot upload images.

    Click Here for Example demonstration

    POST request

    In this method we have security for data.
    This method can send bulk of data.
    It can carry encrypted data from client to server.
    Notes:-
    Post requests cannot be cached.
    Post requests do not remain in browser history.
    Post requests cannot be bookmarked.
    Post requests can be deal with sensitive data.
    Post requests have no limit for data.
    Post requests have good performance.
    Post requests can upload images.

    HTML Tables

    Tables are used to represent data in tabular form.
    Tables are created using <table> tag.
    Table rows are represented using <tr> tag.
    Table cells (data) represented using <td> tag.
    Table is divided into two section head and body.
    Table head is represented using <thead> tag.
    Table body is represented using <tbody> tag.
