Typescript

  • Typescript is a programming language.
  • Typescript is introduced by microsoft.
  • First stable release was 0.8 -> 1st october 2012.
  • Current stable release is 4.5.4 -> 13th December 2021.
  • Typescript is superset of Javascript.
  • Typescript is used to develop angular, react, node, etc applications.
  • Typescript supports OOPs.

    • Environmental setup

  • Download and install NodeJS.
  • Download and install git.
  • Download and install VSCode.
  • Install typescript using following command
  • >npm install -g tsc
  • >npm install -g typescript@latest --force
  • Where npm -> Node packaging manager
  • -g -> global installation
  • tsc -> typescript compiler
  • >tsc -v

    • Transpilation

  • Generally webpages wont understand typescript.
  • as a developer we must convert typescript program to equivalent javascript program.
  • converting one type of program to another type is called as transpilation.
  • 'tsc' is the typescript compiler used to perform transpilation.

    • Execution

  • sample.ts
  • >tsc sample.ts
  • o/p sample.js

    Function

  • A particular business logic is called as function.
  • Functions are used to reuse business logic.
  • We have two types of functions
  • named functions
  • anonymous functions

    • Named functions

  • The function with name is called as named function.
  • We can declare named function using 'function' keyword.
    • Syntax

    function ([arguments_with_type]) : return type
